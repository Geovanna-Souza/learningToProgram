Quarto exemplo

Nessa outra forma de selecionar cores temos o método de hsl, onde as mesmas são selecionadas através de 3 parâmetros, sendo eles: matis(h), saturação(s) e luminosidade(l).

Aplicando gradientes

Para aplicar gradientes devemos criar uma folha de estilo com a propriedade chamada "background-image" nela colocamos a função "linear-gradient" (que também pode ser de outra forma como 'circle'), dentro dessa função inserimos os parâmetros a serem usados, o 1° parâmetro é referente ao ângulo de inclinação do gradiente (pode ser aplicado em graus ou por escrito) e os parâmetros seguintes são as cores a serem usadas.

Criando uma paleta de cores

Para criar uma paleta de cores, podemos usar o Adobe colors (tendo suas limitações pois para salvar a paleta criada é necessário ser assinante), nele podemos escolher o tipo de harmonia que irá utilizar e pegar o código das cores da paleta e aplicar via hexadecimais.
Além do círculo cromático, nesse site é possível extrair cores a partir de imagens na aba "extrair tema" e editar da melhror maneira. Também podendo criar degradê com as cores da imagem.

Outro site onde é possível criar paletas, é o Paletton. Nele podemos fazer o mesmo que no anterior, porém seu diferencial é um simulador onde mostra as cores da paleta aplicada em um site

Capturando cores

Podemos capturar cores através do uso do GIMP, basta seguir os seguntes passos:

  1. Tirar um print da cor desejada;
  2. Abrir a imagem no GIMP;
  3. Selecionar o conta gotas;
  4. E abrir o círculo cromático para pegar o código da cor.

Podemos também instalar uma extenção no chorome (na própria loja do chrome) chamada "colorzilla".
