Sombreamento de caixas

Sombras são muito importantes para dar um efeito de volume e até um leve sensação de realidade. Para aplicar esse efeito em caixas, é necessário utilizar a declaração "box-shadow" em CSS.
Nesse paragráfo podemos visualizar a presença de sombra. É importânte lembrar que não se recomenda o uso de sombras coloridas apenas sombras pretas são recomendadas. Além disso o uso de transparência na cor da sombra é altamente aconselhável.

Propriedades da sombra

A declaração em CSS "box-shadow" apresenta 4 valores a serem declarados, sendo eles:

  • Deslocamento horizontal (h-offset): Que representa o quanto a sombra 'andar' para o lado direito (podendo ser movida para esquerda quando tal valor for negativo);
  • Delocamento vertical (v-offset): Representa o quanto a sombra se desloca para baixo (podendo ser movida para cima quando tal valor for negativo);
  • Embaçamento (blur): Valor relacionado ao quanto a sombra irá se espalhar como fundo;
  • Cor (color): Cor da sombra. Pode usar transparência.
                
                EXEMPLO: box-shadow: 4px 4px 2px rgba(0, 0, 0, 0.315)

Sombras em textos

Existe também uma propriedade específica para criar sombras em textos, denominad text-shadow, que funciona de forma idêntica a 'box-shadow'. Serve principalmente para destacar a letra e seu fundo, dando contraste entre elas.

Bordas decoradas

Como temos a possibilidade de decorar qualquer elemento com as CSS, com as bordas não seria diferente. Podemos arredondar os seus vértices usando a declaração "border-radius".

Propriedades das bordas

As propriedades da bordas podem ser declaradas de várias maneiras diferentes:

  • "border-radius: 10px": Informa que os 4 vértices teram o mesmo valor de raio;
  • "border-radius: 10px 20px 30px 40px": Informa o valor de cada vértice. A ordem se inicia no vértice superior esquerdo;
  • "border-radius: 10px 30px": Informa o valor intercalado do raio dos vértices, sendo o primeiro valor aplicado no vértice superior esquerdo e no vértive inferior direito, e o segundo valor é aplicado nos vértices superior direito e inferior esquerdo.

EXTRA: Bordas personalizadas

Para criar essa bordas, é necessário fazer operarmos no gimp. Nele iremos criar um novo arquivo com as dimenções de 80x80, 70 de resolução e cor de fundo transparente. Em seguida a borda é desenhada e adicionada via CSS no documento HTML através da declaração "border-image-source: url('nome-imagem')" onde a imagem criada é anexada.
Após a imagem ser anexada (e após ter criado uma borda normal), é necessário adicionar outras duas declarações que são as "border-image-slice: numero" e "border-image-repeat: repeat"
