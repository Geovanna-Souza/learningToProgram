Aprendendo e aplicando 'id' e 'class'

Sobre 'id'

Ao criar um documento em HTML, podemos identificar determinado elemento com a função 'id', na folha de estilo, para aplicarmos os estilos para aquele elemento identificado de forma diferente dos demais utilizamos a hashtag (#) entre o tipo do elemento e sua identificação.
Ex: 

                    @charset "UTF-8";
                    h1{
                        color: #D907A9;
                        text-align: center;
                        text-shadow: 2px, 2px, 3px, #F8074A;
                    }
                    
                    h1#principal{
                        color: #8007D9;
                        text-align: left;
                        text-shadow: 2px, 2px, 3px, #5107F8;
                    }

Para que serve o 'id'?

Um 'id' vai identificar um elemento único dentro da página. Essa identificação permite com que seja possível criar estilos especiais para um elemento isolado. Sendo assim, dentro de um documento HTML só se pode ter um único 'id'.

Sobre 'class'

Já na classe podemos classificar determinado(os) elemento(os) com a função 'class', para aplicarmos os estilos para aquele ou mais elementos, nós os identificamos de forma diferente dos demais utilizando um ponto (.) + nome dado a classe dentro do documento em CSS.
Ex: 

                    h2{
                        color: #D907A9;
                        text-align: right;
                        font-size: 1.8em;
                    }
                    
                    .subject{
                        color: #CE05F0;
                        font-size: 1.8em;
                    }
                    
                    .meaning{
                        color: #F8074A;
                        text-decoration: underline;
                    }
                    
                    .obs{
                        color: rgba(0, 0, 0, 0.342);
                    }

Para que serve as 'class'?

As classes vão classificar um ou mais elementos que pertencem ao mesmo grupo, assim compartilhando características em comum a todos os que façam parte do grupo.

Podemos usar a tag <span> Para aplicar os estilos de classes no meio dos textos. Podemos também aplicar um 'id' e uma 'class' no mesmo trecho. Além disso podemos aplicar mais de uma 'class' ao mesmo conteúdo do documento, basta adicionar um espaço entre uma classe e outra.

Pseudo-classes e Pseudo-elementos

Pseudo-classes

As pseudo-classes estão relacionadas ao estado de um elemento ou a uma classe. São representadas em CSS3 pelo símbolo de dois pontos (:). Segue uma lista de algumas das pseudo-classes:

  • :hover (estilo aplicado para quando o mouse é passado por cima);
  • :visited (estilo aplicado sobre links já visitados);
  • :active (estilo aplicado para dar resposta visual a um click);
  • :checked;
  • :empty;
  • :focus.

Pseudo-elementos

São folhas de estilo que podem agir no conteúdo das classes, elementos, e 'id'. Em CSS psceudo-elementos são representados por 2 dois pontos (::). Segue abaixo uma lista de alguns pseudo-elementos:

  • ::before (estilo aplicado antes todo o conteúdo da tag/classe/id em questão);
  • ::after (estilo aplicado após todo o conteúdo da tag/classe/id em questão);
  • ::first-letter;
  • ::first-line.
