Imagens e direitos autorais!

Para utilizar imagens no seu site é necessário atentar-se sobre a questão de direitos autorais.
No Google® no setor de imagens existe um botão escrito "ferramentas" nele é possível filtrar as imagens cujo em teoria são livres de direitos autorais.
Além do Google® também existem outros sites cujo o conteúdo por imagem livre de direitos autorais é fornecido, segue a listra abaixo:
Unsplash;
Pexels;
FreePik;
Pixabay;
Libreshot;
Wikimidia Commons, dentre outros.

Existem vários formatos de imagem, dentre esses formatos os mais utilizados são JPEG e PNG.

O JPEG é um formato de imagem utilizado para gerar imagens extremamente reduzidas

Já o PNG é um formato compacto (porém nem tanto quanto o JPEG), sua principal caracteristica é a capacidade de se configurar opacidade gerando trasparência!

Imagens de Fundo

Podemos adicionar imagens como tema de fundo da nossa página web.
Para isso é necessário utilizarmos a seguinte estrutura de tags:
background-image: url('caminho da imagem');


Entretanto nem todas imagens serão ou ficaram de forma adequada/requerida. Por conta disso é possível personalizar a aplicação da imagem no fundo:
background-size: 100px 'cover' 'contain';
background-repeat: 'no-repeat' 'repeat' 'repeat-x' 'repeat-y';
background-position: 'center center';
background-attachment: 'scrool' 'fixed';
Podemos também compactar e simplificar todas essas propriedades com a shorthand 'background', onde iremos declarar todas as propriedades acima exceto a background-size. Sendo assim declaramos da seguinte maneira: background: color image position repeat attachment;
