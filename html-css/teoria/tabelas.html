Hierarquia de tabelas

Para iniciar a montagem de uma tabela utilizamos a tag <table>. Dentro dela existe uma hierarquia a ser seguida para a montagem completa da tabela:

            
                <table>
                        <tr>TABLE ROW</tr>
                            <th>TABLE HEADER</th>
                                <td>TABLE DATA</td>
                </table>

Uma tabela é composta por esse 3 componentes: O table row= linha de tabela; o table header= cabeçalho de tabela; e o table data= dados de tabela.

Para deixar a tabela com 'cara de tabela', estilizamos a mesma com CSS com os atributos que já estamos acostumados (bordas, acochoamento, fonte, margens entre outras), porém, um novo atributo a ser utilizado é o border-collapse, que serve para deixar a tabela com as células coladas.

Alinhamento do conteúdo de tabelas

O alinhamento do conteúdo presente na tabela é feito de duas formas: Alinhamento horizontal e alinhamento vertical.
O alinhamento horizontal é simples e consiste em utilizar o atributo 'text-align:left/center/right'. Já o alinhamento vertical é feito pela propriedade 'vertical-align:top/middle/bottom'.

Anatomia das tabelas

Geralmente, em tabelas maiores, há uma anatomia básica para construir as mesmas, essa estrutura é composta por: Cabeçalho (Table Head - thead), Corpo (Table Body - tbody) e Rodapé (Table Footer - tfoot).

Caption e escopo de título em tabelas

O caption é a 'legenda da tabela', o título. Já o escopo é aplicado para dizer se o <th> é o título da coluna ou linha correspondente.

            
                <th scope="col"> = Para colunas
                <th scope="row"> = Para linhas

Mesclagem de células

Para mesclar células é necessário inserir os atributos 'colspan' para mesclar colunas e 'rowspan' para mesclar linhas.

            
                <th colspan="3"> = Para colunas
                <th rowspan="3"> = Para linhas

Efeito zebrado

Para melhor visualização das informações pelo usuário (principalmente em tabelas grandes), o efeito zebrado pode solucionar esse problema visual. Para aplica-lo em uma tabela precisamos estilizar a tabela com uma pseudo-classe chamada :nth-child(VALORn) ficando da seguinte maneira:

            
                tbody > tr:nth-child(2n){
                    background-color: var(--cor1);
                }

Essa pseudo-classe também pode receber como valor o 'odd' (ímpar - 2n-1) ou o 'even'(par - 2n).

Fixando o cabeçalho e personalizando colunas

Novamente, trata-se de um recurso para facilitar a experiência do usuário, fazendo com que o cabeçalho permaneça no topo da página, mesmo já tendo sido rolado a muitas linhas atrás. Para isso é necessário ter a tabela com uma 'position:relative;' e fazer com que o 'thead' se sobresaia em relação as 'tr' e 'th', aplicando uma 'position:sticky;'. Porém, somente isso não é o suficiente, já que é preciso indicar que região da tabela nós queremos que seja 'colada' na tela, com isso adicionamos a propriedade 'top:0;' para referenciar a parte de cima da tabela. Por fim, isso apenas carrega o conteúdo das células selecionadas, sendo assim é necessário colorir o fundo com um 'background-color'

            
                table{
                    margin: auto;
                    border: 3px dashed var(--cor2);
                    max-width: 500px;
                    font-size: 1.5em;
                    padding: 5px;
                    background-color: var(--cor3);
                    text-align: center;
                    position: relative;
                }
                thead > tr > th{
                    position: sticky;
                    top: 0;
                    background-color: var(--cor2);
                }

Outra coisa complicada de se fazer era personalizar colunas, porém no HTML5,foi criada a tag <colgroup>, que perminte com que se identifique cada coluna da tabela e possibilita personalizar cada uma sem precisar adicionar um estilo de forma manual em cada uma das células das mesmas.

            
                <table>
                    <colgroup>
                        <col class= "coluna1">
                        <col class= "coluna2">
                        <col class= "coluna3">
                        <col class= "coluna4">
                        <col class= "coluna5">
                    </colgroup>
                    <caption>Título da tabela</caption>
                    <tr>
                    <th scope="col">Nome</th>
                    </tr>
                    ...
                </table>

Responsividade das tabelas

Para evitarmos rolagem lateral em toda página, podemos incluir toda tabela dentro de uma <div>, adicionar um ID em CSS, configurar o atributo overflow/overflow-x/overflow-y com configurações automáticas.

            
                div#container{
                    overflow:auto;
                }
