Links e Âncoras

Para criar-se hyperlinks é necessário criar âncoras através da tag <a>. A principal caracteristica dessa tag é seu parâmetro href= que cria uma referência hipertexto, outro parâmetro muito útil é o hreflang= que permite indicar o idioma principal do site, o que permite que o navegador e/ou softwares de tradução possão traduzir a página caso seja o desejo do usuário.

Manuseando links externos

Links externos são aqueles que redirecionam o internaulta para outro site. Um exemplo desse tipo de link é o link acima.

O problema de realizar a âncoragem desses links externos é que dessa maneira o usuário vai sair do site e iniciar uma nova navegação, o que não é tão bom, sendo assim é recomendado o uso do parâmetro target= contendo o "_blank" e o parâmetro rel= contendo "external". Isso serve para que o link adicionado no site não seja sobreposto sobre o próprio site, assim abrindo uma nova aba com o novo conteúdo.

Exemplo: Esse link contém a nova música da Adele

Target= e seus destinos

Nesse parâmetro podemos usar diferentes tipos de destinos/comandos, segue eles abaixo:
"_self" → Abre o link na janela ou frame atual (padrão);
"_blank" → Abre o link em uma nova janela em branco;
"_top" → Vai desfazer todos os frames e abrir o destino no navegador completo;
"_parent" → Similar ao uso do "_top", em uma referência à janela mãe nome-do-frame caso esteja usando frames, indicar o nome da janela a abrir.

Rel= e seus recursos

Esse parâmetro tem o intuito de apontar a natureza do destino do link, sendo representados pelos seguintes destinos/comandos:
"next" → Indica que o link é para a próxima parte do documento atual;
"prev" → Indica que o link é para a parte anterior do documento atual;
"author" → Indica que é um link para o site do autor do artigo atual:
"external" → Indica que é um link para outro site que não faz parte do site atual;
"nofollow" → Indica que o site não é 100% confiável.

Manuseando links internos
