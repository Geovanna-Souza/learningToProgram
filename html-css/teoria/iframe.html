Iframe e suas utilidades

Iframes são um "frame inline" que projetam/reproduzem um quadro online, ou seja, uma jenela que projeta outra página web dentro da pagina atual

Testando o iframe

o iframe é composto principalmente por uma 'source' que contém a url o site externo a ser exibido, um 'frameborder' que deixa ou não as bordas da caixa e o 'title' que serve para identificar o iframe em questão.


Observação: Como alguns dispositivos de celular não suportam o iframe, é interessante deixar uma mensagem ao usuário e/ou um link para redirecioná-lo

Um iframe por padrão tem suas dimenções de exibição em 350x150 pixels, mas isso pode ser editado individualmente ou nas CSS que são soberanas em relação ao método anterior.

Barra de rolagem

Podemos também adicionar, ou não, barras de rolagem nos iframes através do parâmetro 'scolling' que recebe os seguintes valores: auto - é o padrão e quando o conteúdo excede o tamanho do frame ele automáticamente adiciona a barra de rolagem; yes - adiciona a barra de rolagem mesmo sem o conteúdo exceder o tamanho do frame; e o no - não permite rolagem no frame.

Iframe ou DIV?

A maior vantagem do iframe, é a exibição de uma página com outra estrutura, outros estilos, outro design sem ter que alterar tudo isso na página local. Isso é muito vantajoso para exibir páginas locais no atual documento.
Caso contrário, caso o conteúdo seja diretamente relacionado ao contexto sendo descrito na página, a div provavelmente é a melhor opção.
Observação: O iframe gera um problema de indexação e relacionamento do conteúdo do site pelos mecanismos de busca quando usado, por conta disso é importânte avaliar quando deve- usado.

Navegação com o Iframe

Com o iframe podemos exibir mais de uma página no mesmo frame, assim tornando seu uso mais interessante. Para isso é necessário conectar os links âncoras com o iframe, direcionando o target da âncora para o iframe que tem que ser identificado de alguma maneira = receber um parâmetro 'name' onde será atribuído um nome para o iframe que exibirá o conteúdo do link e/ou página local ou do mesmo repositório.
