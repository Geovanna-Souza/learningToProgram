Responsividade e variáveis em CSS

Largura e altura adaptáveis

Um site responsivo é capaz de se adaptar de forma visível a diferentes tamanhos de telas. Para que nosso site se adapte mais facilmente ao tamanho da tela, uma das técnicas básicas que podem ser utilizadas é deixar algumas caixas estrategicamente um pouco mais flexíveis indicando valores mínimos e máximos.

            
            exemplo-box{
            min-width: 400px;
            max-width: 800px;
            min-height: 1000px;
            max-height: 2000px;
            }

Essa configuração vai garantir que a caixa em questâo não fique com mais larga que 400px (para ser melhor visualizada em telas pequenas) e nem seja maior que 800px (para telas maiores).
Para medidas entre 400px e 800px, o tamaho será adaptável e a caixa vai ocupar 100% da largura do contêiner. Isso vai facilitar bastante a adaptação do conteúdo em diversos tipos de tela.
OBS: É possível aplicar técnicas adicionas para dar ainda mais capacidade de adaptaçãodo conteúdo com as mediasqueries, com a regra @media e as caixas flexíveis como as flexboxes.

Variáveis em CSS

Para criarmos variáveis em CSS, devemos definir uma área de definiçoes dentro do seu estilo para uma pseudo-classe chamada :root, que define as configurações para cada 'raiz de uma árvore' (vulgo a página), que se aplica a todo o documento por padrão. Para declarar uma atribuição na root, descrevemos da seguinte forma: --nomeAtributo:valor;

            
            :root{
                --corDeFundo:#c5ebd6;
                --fonteTitulo: 'Arial, Helvetica, sans-serif';
                --bordaSecundária: border-image-source:url(img.png);
            }

A partir disso, definir cores e fontes no documento HTML, fica mais fácil e personalizavel, utilizando a função var()

            
            body{
                font-family: var(--fonteTitulo);
                color:var(--corDefundo);
                background-color:var(--cor2);
            }

Uso do seletor * em CSS

O asteristico (*) tem uma função muito importante e especial, pois basicamente ele aplica uma configuração padrão para todos os elementos do código HTML ao qual o estilo está sendo aplicado.
Esse seletor global pode ser usado principalmente para eliminar as eventuais margens que os navegadores adicionam a alguns elementos (user agents), permitindo a personalização de medidas que irão ser exibidas na tela de cada elemento de forma individual.
