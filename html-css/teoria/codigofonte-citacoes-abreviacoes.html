Código-fonte

Dentro do mundo da programação códigos são compartilhados a todo tempo em multiplas linguagens, por conta disso existe a tag <code> da HTML, onde delimita a região que contém código de alguma linguagem de programação. Sua principal vantagem semântica, é que essa tag indica ao navegador que tal trecho se trata de um código de computador (exibindo efeito visual tornando as letras monospace).

let numbers = [5, 3]; function ArrayOrdem(){ console.log(numbers.reverse()) } ArrayOrdem();

Porém, somente essa tag não fornece a formatação do código (pois o mesmo fica extendido por toda a linha de forma contínua). Para solucionar tal situação utiliza-se a tag <pre> para tabular o código.

Segue o exemplo: 

                
            let numbers = [5, 3];
            function ArrayOrdem(){
            console.log(numbers.reverse())
            }
            ArrayOrdem();

Citações

Citação simples

Para realizar citações em HTML usa-se a tag <q>, e o texto que estiver dentro dessa tag recebe automáticamente aspas duplas (não havendo deslocamento).

Exemplo: Segundo Nelson Mandela A maior glória de viver não está em nunca cair, mas em nos levantar toda vez que caímos.

Citação em bloco

Utilizado para realizar citações mais longas e que tenham um parafrágo só para si. Para realizar tal ação usa-se a tag <blockquote>,que faz com que o texto recue automáticamente, podendo também colocar um link para o texto original, usando o parâmetro cite dentro da tag.

Exemplo: ... podemos ver isso em uma passagem do livro "A mente em paz, de tudo é capaz" de Danilo H. Gomes, onde o autor diz:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Nostrum, sequi nemo ipsa explicabo dolores repudiandae quod qui ratione temporibus illo eligendi iure possimus ad, non eaque, at eius sapiente reprehenderit?

Abreviações

Utilizada para abreviar as palavras a tag <abbr> pode abreviar frases/palavras, deixando claro para o usuário e os mecanismos de busca o significado da mesma. Dentro do title se insere a palavra a ser abreviada.

Exemplo: Podemos abreviar o nome da série "Game of thrones" para GOT

Texto invertido

Essa funcionalidade é operada pela tag <bdo>, que possui um parâmetro (dir) responsável por direcionar o sentido do texto a ser invertido, sendo dois sentidos possíveis: RTL = right-to-left, e LTR = left-to-right.

Exemplo: Quero ficar rica logo!; Quero ficar rica logo
