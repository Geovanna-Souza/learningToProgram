Usando as CSS3

Aqui iniciaremos o manuseio de estilos dentro do site.

O parâmetro STYLE (In Line)

Esse parâmetro fornece estilos de fonte e cor, entretanto é necessário atentar-se com a forma que esse parâmetro é preenchido, tendo-se assim muita atenção com letras maíusculas e separação com ponto e vírgula (;). Nesse paragráfo temos uma aplicação das CSS 'in line', essa aplicação é feita de maneira exclusiva para o título em questão (não irá aplicar a todos os h2 nesse caso). Por conta disso esse método não é usado no dia-a-dia, e sim para realizar modificações muito pontuais. 

                
                    <h2 style="font-family: Arial, Helvetica, sans-serif; color: rgba(255, 0, 255, 0.473);">O parâmetro STYLE (InLine)</h2>

Estilos locais / internos

Para realizarmos tal tipo de estilização temos que criar uma tag <style> dentro da tag <head>, e a partir de lá podemos aplicar estilos que irão ser executados em todo o conteúdo da página ATUAL. Segue um exemplo: 

                <head>
                    <style>
                        body {
                            background-color: black;
                            font-size: 15px;
                            font-family: Georgia, 'Times New Roman', Times, serif;
                            color: white;
                        }
                        h1 {
                            color: white;
                        }
                        h2 {
                            color: red
                        }
                    </style>
                </head>

CSS external style

Esse tipo de aplicação de estilo é o mais versátil que tem, nele podemos aplicar e editar as CSS em diversas páginas diferentes sem precisar entrar em cada uma delas para edita-las individualmente.
